NAME
====

File::Path::Expand - expand filenames

SYNOPSIS
========

``` raku
use File::Path::Expand;

say expand-filename( '~/path/to/hoge.txt' );
```



INSTALLATION
============

まだ ecosystem に追加してないので:

    $ zef install https://github.com/noqisofon/raku-File-Path-Expand.git

でインストールできます。
